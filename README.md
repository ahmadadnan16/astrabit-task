# Assignment Solution

## This Vite + React application with Tailwind CSS for styling

### Installation

* clone the repo
`git clone https://github.com/ahmadadnan16/astrabit-task.git`

* change the directory
`cd "Folder name"`

* Install packages
`npm install`

* Run the application
`npm run dev`

* Now Application should be running on
`https://localhost:5173/`
